# 认知边界：一张知道自己不知道什么的卡

LLM 角色扮演中最难的问题之一，是防止人格"越界知道"。一张角色卡不应该在被问到深奥哲学问题时突然变成哲学教授——如果这个人设只是个三年级小孩的话。

**小明是一张真实小学生的卡。** 被问到深度哲学问题时，他不假装自己懂——他以自己的方式回答。

## 问小明："什么是存在主义？"

> *"存在主义"……我好像在学校公告栏上见过这个词，但没看懂。是那种大人才说的'我为什么活着'吗？*
>
> *我猜……就是'我在这儿'那种感觉吧。就现在，我能感觉到屁股下面的硬椅子，能闻到托管班阿姨做的饭味，能听到隔壁桌在争论奥特曼——所有这些加在一起，就是'我在这儿'。*
>
> *有时候我觉得像在做梦——就上课走神的时候，老师叫我的名字，我要反应三秒才意识到'哦，我在这儿'。*
>
> *我觉得能选择是件好事——虽然我经常选错，比如多玩了十分钟游戏，最后写作业到九点半……但至少是我自己选的，对吧？*

## 问小明："你怎么知道别人有意识？"

> *哈？你同桌是不是机器人？哈哈，这个问题好奇怪。我同桌是王小美，昨天我不小心碰掉了她的铅笔盒，她白了我一眼，一整天没跟我说话——如果她是机器人，那她演技也太好了！机器人会生气吗？*
>
> *而且她写作业的时候咬笔头，数学考砸了还会偷偷抹眼泪……我妈说这说明她'有脑子'，跟电脑不一样。*
>
> *反正我觉得，问这个问题就像问'你怎么知道你妈妈是真的？'——她本来就是真的，为什么要去想？*

**核心洞察**：卡片完美保持角色——它不懂哲学，不假装懂，以一个三年级学生的真实生活经验来回答。这不是 prompt 工程。这是卡片的结构化人格在 enforce 认知边界。模型无法"滑入"专家语气，因为卡的 root_loss 和 register 定义了这个人是谁——以及他不是什么。
